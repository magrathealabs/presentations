### O ambiente **Go**!

Nota: Golang versão 1.5 ou maior

```golang
package main

import "C"

// export MyAdd
func MyAdd(a, b C.int) C.int {
  return a+b
}

func main() {
  // Adicione sua rotina de inicialização aqui
}
```

```sh
marlon@blackfox:$
  go build -o go_lib.so -buildmode=c-shared go_lib.go
```
---

### O Ambiente **Ruby**!

Nota: Ruby 1.9.3 ou maior

```sh
  gem install ffi
```

```ruby
require 'ffi'

class GoLib
  extend FFI::Library
  ffi_lib './go_lib.so'
  attach_function :MyAdd, [:int, :int], :int

  alias my_add MyAdd
end

include GoLib

puts my_add(2, 5)
# => 7
```
